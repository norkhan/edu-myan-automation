# Educate Myanmar Automation
 

### OS
```
Ubuntu 18.04.4 LTS
```

### Environments Setup

| Name  | Version |
| ------------- | ------------- |
| node  | v8.10.0  |
| npm  | 6.14.4  |


### Installing

Run 'start.sh' script that will install all dependencies and packages and
start web-automation

#### Automate to Production

```
chmod +x start.sh && ./start.sh
```

#### Automate to Development

```
./start.sh dev
```

### Runnig the tests

```
npm test
```
